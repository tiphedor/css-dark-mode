prefers-color-scheme media query

A new CSS draft introduces the prefers-color-scheme media query, allowing website designers to write UI that reflect the user preferences on color scheme.

Usage:

@media (prefers-color-scheme: light) {
// Light mode code
}
@media (prefers-color-scheme: dark) {
// Dark mode code
}
